# Years Away

This project matches a birth date to a local-catalog star whose light-travel distance is approximately equal to the user's age, then renders a spectrum-inspired visual and story.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Start the frontend and API together:
   `npm run dev`
3. Open:
   `http://localhost:3000`

## API

`POST /api/star-match`

`GET /api/star/:id/details`
